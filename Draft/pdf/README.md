# PDF Drafts

Esta carpeta agrupa borradores de documentos PDF antes de llevarlos al pipeline real de generación.

## Objetivo

Permitir definir:

- layout visual
- estructura de secciones
- encabezados, pies y bloques de firma
- contenido obligatorio por tipo de documento

## Tipos previstos

- `actas-entrega/`
- `actas-recepcion/`
- `shared/` para bloques reutilizables
