# Mail Drafts

Esta carpeta agrupa borradores de correos antes de convertirlos en plantillas reales del sistema.

## Objetivo

Permitir validar:

- asunto
- tono
- estructura del cuerpo
- llamados a la acción
- datos variables por tipo de evento
