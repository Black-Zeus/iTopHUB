# Template Governance

Portable governance starter for another repository.

## Includes

- `AGENTS.md`
- `REGENERATE.md`
- `docs/README.md`
- example files for `containers`, `domains`, and `operations`

## How To Use

1. Copy this folder into the target repository.
2. Move or merge `AGENTS.md` and `REGENERATE.md` to the repository root if needed.
3. Keep `docs/README.md`.
4. Replace the example files with real project-specific documents.
